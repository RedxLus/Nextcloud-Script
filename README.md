## Instalación

Necesita ejecutar el Script con permisos root. Para ello segun el sistema puede hacerlo de varias maneras:

```
sudo sh menu.sh
```

Tambien entrando diretamente como usuario root y ejecutando el script:

```
su
```
```
sh menu.sh
```
